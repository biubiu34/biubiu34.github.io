​ 博客中只有文字和图片感觉还是太枯燥了，就想到了能不能在hexo博客中加入音乐和视频。尝试了用网易云音乐外链，操作简单但是很多都因为版权问题不能播放。然后我查到了aplayer和dplayer这两个hexo插件。

1.在hexo插入音乐

1.使用网易云外链播放器（可能有版权问题）

​ 使用网易云外链方法可以说很简单，但是很多因为版权问题不能生成外链播放器。

遇到问题：

  • 在markdown和博客页面中都能显示图标但是播放不了音乐。
  • 猜测可能是版权问题

无意间看到别人的博客，粘贴了幻听的代码之后，发现“幻听”又是可以播放的。

2.使用

​ 安装完成后，在需要的地方添加

​ 标签参数：

  • title : 曲目标题
  • author: 曲目作者
  • url: 音乐文件 URL 地址
  • picture_url: (可选) 音乐对应的图片地址
  • narrow: （可选）播放器袖珍风格
  • autoplay: (可选) 自动播放，移动端浏览器暂时不支持此功能
  • width:xxx: (可选) 播放器宽度 (默认: 100%)
  • lrc:xxx: （可选）歌词文件 URL 地址

2.然后就可以通过 这个标签meting 在文章中使用 MetingJS 播放器了

  • 有关选项列表如下:
选项 默认值 描述
id 必须值 歌曲 id / 播放列表 id / 相册 id / 搜索关键字，
server 必须值 音乐平台: netease, tencent, kugou, xiami, baidu
type 必须值 song, playlist, album, search, artist
fixed false 开启固定模式
mini false 开启迷你模式
loop all 列表循环模式：all, one,none
order list 列表播放模式： list, random
volume 0.7 播放器音量
lrctype 0 歌词格式类型
listfolded false 指定音乐播放列表是否折叠
storagename metingjs LocalStorage 中存储播放器设定的键名
autoplay true 自动播放，移动端浏览器暂时不支持此功能
mutex true 该选项开启时，如果同页面有其他 aplayer 播放，该播放器会暂停
listmaxheight 340px 播放列表的最大长度
preload auto 音乐文件预载入模式，可选项： none, metadata, auto
theme #ad7a86 播放器风格色彩设置
