Nginx的安装

  • yum 安装

    • 使用epel源安装

      • [root@RouterB ~]# yum install epel-release -y
      • [root@RouterB ~]# yum install nginx -y

    • 使用nginx官方源安装

  • 编译安装

    • 下载源码

    • 解压安装

      • 编译安装3步：

        ./configure –prefix=/usr/local/nginx8

        make

        make install

      • 通过命令启动和关闭nginx

        命令 作用
        nginx 启动服务
        nginx -s reload 不停止服务重读配置文件
        nginx -s stop 停止服务
